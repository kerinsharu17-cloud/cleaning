# Titanic Dataset - Data Cleaning and Exploratory Data Analysis (EDA)

## Objective
Perform data cleaning and exploratory data analysis (EDA) on the Titanic dataset to identify patterns, trends, and relationships between different variables affecting passenger survival.

## Dataset
The dataset used is the Titanic dataset containing passenger information such as age, gender, ticket class, fare, and survival status.

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

## Data Cleaning
The following preprocessing steps were performed:

1. Checked for missing values.
2. Filled missing values in the `Age` column using the mean age.
3. Filled missing values in the `Embarked` column using the mode.
4. Removed the `Cabin` column due to excessive missing values.

## Exploratory Data Analysis
The following analyses were performed:

### 1. Survival Count
A bar chart was created to visualize the number of passengers who survived and those who did not survive.

### 2. Gender vs Survival
A comparison was made between passenger gender and survival status.

### 3. Age Distribution
A histogram was used to analyze the distribution of passenger ages.

## Findings
- Female passengers had a higher survival rate than male passengers.
- Most passengers were between 20 and 40 years of age.
- Passenger class and gender significantly influenced survival.
- Data cleaning improved the quality and usability of the dataset.

## Conclusion
The Titanic dataset was successfully cleaned and analyzed. Exploratory Data Analysis helped identify important patterns and relationships in the data, particularly the impact of gender, age, and passenger class on survival rates.
